# Typescript Rest Server

- Inicio..

## Npm Install

      npm i typescript
      npm i tslint
      npm i express
      npm i cors
      npm i dotenv

## Configuracion

- tsconfig
      Crear el tsconfig.json  node_modules/.bin/tsc --init
      "outdir": "./dist"
      "moduleResolution": "node"
      "sourceMap":true

- tslint
      Crear tslint.json - node_modules/.bin/tslint --init
      "no-console": false

- tsc ( Modo observador )
      "build": "tsc --watch"
